# Paytm Phishing Email Analysis
## Overview
This project focuses on analyzing a phishing email that impersonates Paytm. The objective is to identify phishing indicators and understand how attackers use social engineering techniques to steal user credentials.
## Objectives
•	Analyze a suspicious email.
•	Identify phishing characteristics.
•	Understand common phishing techniques.
•	Improve cybersecurity awareness.
###Tools Used
•	Email Inspection
•	URL Analysis
•	Manual Phishing Detection Techniques
### Key Findings
•	Suspicious sender email address
•	Fake Paytm verification link
•	Urgent account suspension warning
•	Generic greeting
•	Potential credential theft attempt

Conclusion
The analyzed email was identified as a phishing attempt designed to trick users into revealing sensitive information. Careful inspection of sender details, links, and email content can help prevent such attacks.
